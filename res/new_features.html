 充满惊喜的搜索


V2EX 的 [ 搜索 ] 功能，简直就像是一个小小的 Google！除了可以帮你找到与特定关键词相关的主题及回复外，还可以当作一个词典来用，比如如果你不知道 livid 这个单词是什么意思，你可以执行下面的搜索：

http://www.v2ex.com/q/livid

然后你就会在结果顶端看到对这个单词的解释。而如果你把单词拼错的话，V2EX 搜索还会自动提示你一个或多个接近的正确的拼写。比如你可以试试看：

http://www.v2ex.com/q/perhaaps

有趣吧？尽情享受吧。

如果你是一个专业的 PHP 程序员，你可能还会对 V2EX 特别提供的 [ 参考文档藏经阁 ] 感兴趣，这是一个专门针对 PHP 参考文档的高性能搜索引擎，目前索引了 PHP / PEAR / Zend Framework / Smarty 等 PHP 相关参考文档。为你的 PHP 编程提供帮助。

而前段时间 V2EX 还增加了小小的针对手机号码的搜索引擎 [ 手机号码所在地查询 ]，通过这个小小的工具你可以查到任何一个中国大陆境内的手机号码所在的地区。
